# Polygon Editor

Одностраничное веб-приложение для работы с полигонами на `canvas` без фреймворков.

## Деплой

[grand-tapioca-9a8a70.netlify.app/](https://grand-tapioca-9a8a70.netlify.app/)

## Что умеет приложение

- Генерирует случайные полигоны (3-7 вершин, случайный цвет, случайная позиция).
- Выбирает полигон кликом внутри фигуры.
- Перетаскивает выбранный полигон мышью.
- Ограничивает перемещение границами поля.
- Запрещает наложение полигонов друг на друга.
- Удаляет выбранный полигон (`Delete`) и удаляет все полигоны.
- Поддерживает историю действий: `Undo/Redo` (`Ctrl+Z`, `Ctrl+Y`, `Ctrl+Shift+Z`).
- Показывает информационную панель (количество полигонов и текущее выделение).
- Показывает toast-сообщения для важных сценариев.
- Имеет анимацию появления новых полигонов.
- Дополнительно: смена цвета выбранного полигона, экспорт/импорт сцены в JSON.

## Технологии

- Vanilla JavaScript (ES modules)
- Web Components
- HTML5 Canvas
- CSS Grid / Flexbox
- Webpack 5

## Быстрый старт

### 1) Установка зависимостей

```bash
npm install
```

### 2) Запуск в режиме разработки

```bash
npm start
```

Приложение будет доступно на `http://localhost:8080`.

### 3) Production-сборка

```bash
npm run build
```

Готовые файлы будут в папке `dist`.

## Управление

### Кнопки

- Сгенерировать полигон
- Удалить выбранный
- Удалить все
- Отменить (Undo)
- Повторить (Redo)
- Экспорт JSON
- Импорт JSON

### Горячие клавиши

- `Delete` - удалить выбранный полигон
- `Ctrl+Z` - Undo
- `Ctrl+Y` или `Ctrl+Shift+Z` - Redo

## Как проверить функциональность

1. Нажмите `Сгенерировать полигон` несколько раз.
2. Выберите фигуру кликом и перетащите ее.
3. Попробуйте увести фигуру за границу поля (должно ограничиваться).
4. Попробуйте наложить фигуры друг на друга (не должно получиться).
5. Нажмите `Undo/Redo` после генерации, перемещения, удаления.
6. Нажмите `Удалить выбранный` без выделения (должен появиться toast).
7. Экспортируйте сцену в JSON, очистите поле, импортируйте обратно.
